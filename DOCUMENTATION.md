# Warehouse Management System

## Overview
This is a warehouse management system built with CodeIgniter 4 framework.

## Features
- Inventory management
- User authentication
- Dashboard analytics
- Report generation

## Installation
1. Clone the repository
2. Configure database settings
3. Run migrations
4. Start the server

## Usage
Access the application through your web browser at `http://localhost/warehouse-working-file-main/public`

## Requirements
- PHP 8.1 or higher
- MySQL database
- Web server (Apache/Nginx)
